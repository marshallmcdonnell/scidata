SciData - A Scientific Data Model

SciData is is a data model for scientific data that provides a framework for organizing and annotating both the data and metadata from scientific experiments, calculations, and theories.

SciData Context Documents

Importing context documents in JSON-LD files allows you to use the context for multiple documents. This is the same way that XML (and JSON) schema work except the context is not defining rules of what the content of the JSON values can be, rather assigning semantic meaning to the values.

Below is a list of the current context files developed for the scientific data model. Examples of each of these can be found on the examples page.

  • scidata - Top level JSON-LD context (JSON object) that you need to include in each of your scidata documents
  • JSON-LD Contexts for the main parts of the framework
    • scidata_methodology - object for all the methodology aspects
    • scidata_context - object for all the context facets
    • scidata_dataset - object for all the data (includes datagroup, dataseries, datapoint
    • scidata_parameter - object for representing the metadata of a generic parameter or array of parameter values
    • scidata_value - object for representing the metadata of a value (numeric or text) or array of values
    • scidata_unit - object for representing or referencing scientific units
