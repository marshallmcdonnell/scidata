SciData - A Scientific Data Model

SciData is is a data model (shown right) for scientific data that provides a framework for organizing both the data and metadata from scientific experiments, calculations, and theories.

SciData JSON-LD Context Documents

Importing context documents in JSON-LD files allows their use in multiple documents. This is the same way that XML (and JSON) schema work except the context is not defining rules of what the content of the JSON values can be, rather the assignment of semantic meaning to the values. Below are a list of the context files developed for the scientific data model, with examples. Readers wanting to see how the JSON-LD files convert to RDF should go to the JSON-LD Playground. NOTE: JSON can be best viewed using Firefox and the JSONView extension.

  • scidata - Top level JSON-LD context for each scidata document - examples: NMR Spectrum • pH Measurement • Literature Refractive Index • SCF Calculation
  • scidata_methodology - methodology aspects (Comp. example) (Expt. example)
  • scidata_context - context facets (Comp. example) (Expt. example)
  • scidata_dataset - data (includes datagroup, dataseries, datapoint (example)
  • scidata_parameter - generic parameter or array of parameter values (example)
  • scidata_value - a value (numeric or text) or array of values (example)
  • scidata_unit - representation or referencing scientific units (example)
