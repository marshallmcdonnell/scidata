SciData - A Scientific Data Model

SciData is is a data model (shown right) for scientific data that provides a framework for organizing both the data and metadata from scientific experiments, calculations, and theories.

SciData JSON-LD Context Documents

Importing context documents in JSON-LD files allows their use in multiple documents. This is the same way that XML (and JSON) schema work except the context is not defining rules of what the content of the JSON values can be, rather the assignment of semantic meaning to the values. A list of example files is here and includes data from PubChem, the Dortmund Data Bank (DDB), and a Crystallography Information Framework (CIF) file.

  • SciData Framework JSON-LD Contexts
    • scidata - Top level JSON-LD context for each scidata document
    • scidata_methodology - methodology aspects
    • scidata_system - system facets
    • scidata_dataset - data (includes datagroup,dataseries, datapoint
    • scidata_parameter - generic parameter or array of parameter values
    • scidata_value - a value (numeric or text) or array of values
    • scidata_unit - representation or referencing scientific units
  • Example JSON-LD Contexts for 'Aspects' of Methodology
    • For Experimental Data
      • scidata_measurement - object for measurement metadata
      • scidata_procedure - object for procedural metadata
      • scidata_resource - object for laboratory resource metadata
    • For Computational Data
      • scidata_basisset - object for basis set metadata (example)
      • scidata_calculation - object for calculation metadata (example)
      • scidata_software - object for computational software metadata (example)
  • Example JSON-LD Contexts for 'Facets' of the System
    • For Experimental Data
      • scidata_chemical - object for (manufactured) chemical metadata
      • scidata_compound - object for compound metadata
      • scidata_condition - object for (experimental) conditions metadata
      • scidata_material - object for materials metadata
      • scidata_organism - object for organism metadata
      • scidata_system - object for chemical system (mixture) metadata
    • For Computational Data
      • scidata_computer - object for computer processing metadata
      • scidata_molsystem - object for molecular system metadata
